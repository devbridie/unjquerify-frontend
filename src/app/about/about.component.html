unjQuerify

unjQuerify is a tool developers can use to migrate an existing codebase dependant on jQuery to vanilla JavaScript.

What's wrong with jQuery?

The web has come a long way since the creation of jQuery in 2006. At its inception, the web was fraught with browser incompatibilities. Now, web APIs have evolved to be more standardized and developer-friendly. This removes part of the need for jQuery, and can reduce bloat on webpages, removing unnecessary page loading times, increasing performance, and allowing the web to progress.
